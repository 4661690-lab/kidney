Privacy Policy

Last updated: June 2026

1. Overview

This application ("ZADIRAKAQI Comment Bot") is a private tool used solely by the YouTube channel owner to respond to comments on the ZADIRAKAQI YouTube channel.

2. YouTube API Services

This application uses YouTube API Services. By using this application, you agree to be bound by the Google Privacy Policy: https://policies.google.com/privacy

3. Data Collection

The application accesses the following data via YouTube API:

  • Public comments posted on ZADIRAKAQI YouTube videos
  • Comment author usernames (public information)

4. Data Usage

Collected data is used exclusively to generate and post replies to viewer comments on the ZADIRAKAQI channel. Data is not shared with third parties.

5. Data Storage

Comment IDs and reply texts are stored locally in a JSON file on the operator's private computer. No data is stored on external servers.

6. Data Deletion

To request deletion of any stored data related to your comments, contact: 4661690@gmail.com

7. Revocation of Access

You can revoke this application's access to your Google account at: https://security.google.com/settings/security/permissions
